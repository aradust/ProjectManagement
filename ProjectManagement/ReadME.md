# Руководство пользователя

## Вход в систему

Перейдите в браузере по адресу `http://localhost:3000`. После открытия приложения отобразится форма авторизации.

Введите значение в поле `Email`, затем укажите пароль в поле `Password` и нажмите кнопку `Login`.

![alt text](image.png)

После успешной авторизации откроется главная страница приложения.

![alt text](image-1.png)

В правом верхнем углу расположена кнопка выхода из системы.

## Создание сотрудников

Для начала работы необходимо перейти к разделу `Employees`. Для этого прокрутите главную страницу вниз и откройте соответствующую вкладку.

![alt text](image-3.png)

В разделе `Employees` заполните данные сотрудника, назначьте ему роль `Employee`, затем сохраните введенные email и пароль для дальнейшего входа в систему.

![alt text](image-4.png)

Аналогичным образом создайте учетную запись менеджера.

![alt text](image-5.png)

После завершения нажмите кнопку `Home` в правом верхнем углу, чтобы вернуться на главную страницу.

## Создание проекта

На главной странице нажмите карточку `CreateProject`. После этого откроется первый шаг мастера создания проекта.

На первом шаге укажите название проекта, даты начала и окончания, а также приоритет. Затем нажмите кнопку `Next`, чтобы перейти к следующему шагу.

![alt text](image-6.png)

На втором шаге укажите компанию-заказчика и компанию-исполнителя.

![alt text](image-2.png)

На третьем шаге выберите менеджера проекта.

![alt text](image-7.png)

На четвертом шаге выберите сотрудника, который будет включен в проектную команду.

![alt text](image-8.png)

На пятом шаге загрузите прикрепляемое изображение или иной файл.

![alt text](image-10.png)

После завершения заполнения нажмите кнопку `Finish`. На следующем экране нажмите `Go to Home`.

![alt text](image-9.png)

## Просмотр проекта

После возврата на главную страницу откройте раздел `Projects`, чтобы увидеть созданный проект. Если в системе содержится несколько проектов, для поиска нужного можно воспользоваться фильтрами.

![alt text](image-11.png)

## Создание задачи

С главной страницы также можно перейти в раздел `Tasks`, нажав соответствующую карточку.

После перехода нажмите кнопку `+ New Task`, заполните необходимые данные и нажмите `Create Task`.

![alt text](image-12.png)

После этого будет создана карточка задачи. Задачу можно редактировать. При большом количестве задач можно использовать поиск и фильтры.

## Регистрация нового пользователя

Если выйти из учетной записи, система снова отобразит форму авторизации. На этой странице можно нажать кнопку `Sign Up` и перейти к форме регистрации.

При корректном заполнении данных и последующем входе в систему с указанными учетными данными пользователю будет назначена роль `Employee`.

![alt text](image-13.png)

## Работа в роли сотрудника

После входа в систему под учетной записью сотрудника количество доступных функций будет ограничено.

![alt text](image-14.png)

Пользователь может просматривать только свои проекты.

![alt text](image-15.png)

Если у проекта имеются документы, для их отображения необходимо нажать на стрелку слева от блока `Documents`.

Также сотрудник может просматривать свои задачи и при необходимости изменять их статус.

![alt text](image-16.png)

## Работа в роли менеджера

При входе в систему под учетной записью менеджера главная страница отображается аналогично странице сотрудника.

Менеджер может редактировать проект, добавляя или удаляя сотрудников. Для этого необходимо нажать кнопку `Edit` на карточке проекта.

![alt text](image-17.png)

После внесения изменений следует нажать кнопку `UpdateProject`.

![alt text](image-18.png)

Кроме того, менеджер может изменять статус задач и назначать задачи другим сотрудникам. Для этого необходимо изменить соответствующие параметры в карточке задачи.

![alt text](image-19.png)
